# 🛒 Proyecto de E-commerce API

Este proyecto es una aplicación web tipo e-commerce desarrollada con FastAPI. Permite la gestión de usuarios, productos, categorías y carrito de compras, incluyendo autenticación segura mediante JWT y un panel de administración.

---

## 🚀 Características

- Autenticación de usuarios con JWT (login y registro)
- Panel de administración para gestión de artículos, usuarios y categorías
- Carrito de compras funcional
- Gestión de productos y categorías
- Base de datos compatible con PostgreSQL y SQLite
- Arquitectura preparada para escalabilidad

---

## 🛠️ Tecnologías utilizadas

**Backend:**
- Python
- FastAPI

**Frontend:**
- HTML
- CSS
- JavaScript

**Base de datos:**
- PostgreSQL
- SQLite

---

## 🧰 Requisitos
